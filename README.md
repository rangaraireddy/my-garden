## my-garden app in NATIVESCRIPT

Clone this repo into your local 

`git clone https://github.com/rangaraireddy/my-garden.git`


Now do `npm install`


Run the project with `tns run ios` or `tns run android` depending on what simulator you need to run this which would open the app in that respective emulator.


Debug using this `tns debug ios` or `tns debug android`
